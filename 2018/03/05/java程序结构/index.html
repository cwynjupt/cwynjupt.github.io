java程序结构

Java程序结构

package demo.day01; //必须是小写字母，多个单词用.隔开
import Java.util.Scanner; //java API 一定在当前库存在
public class HelloWorld{ //类名每个单词首字母大写 

//类体中成员要缩进一个tab宽
`public static void main(String [] args){`
//方法中的成员也要缩进1个tab

    `Scanner console=new Scanner(System.in);`
`System.out.println();`

}`

} //类体 class body
//括号要配对，声明开始和结束的位置要对齐

编译错误：Java编译器在编译源代码为class文件的过程中出现的错误，一般为语法错误
当出现编译错误是没有class文件产生，也就不能运行程序。
eclipse 中出现的红色× ：编译错误

在控制台运行Java程序
1.设置CLASSPATH环境变量,指向package所在的目录（bin）
2.执行Java package.classname
