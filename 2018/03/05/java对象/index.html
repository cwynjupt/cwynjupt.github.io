对象

引用本身不是对象，引用指代了对象
引用的值是对象的地址值，通过地址引用对象
引用的值不是对象
引用变量在运行期间动态绑定到对象

变量不是对象，当变量的值是null时，调用了变量的属性或方法会出现空指针异常 ！！！！

构造器（构造方法）

在类中定义

面向对象：
类型：对象的模板，是对象的数据结构定义 —–月饼模子
对象：利用类作为模板创建的个体实例，本质是数据 ——月饼
引用：是对个体的标识名称——-月饼的吊牌
类中可以声明：成员变量，方法，构造器
成员变量：包括实例变量，静态变量
实例变量：用于声明对象的结构，在创建对象时分配内存，每个对象有一份
方法：是用于对当前对象数据进行算法计算，实现相应的业务功能。
方法是对象的功能，对象的动作，对象的行为：总之是动词
构造器：用于创建初始化对象的方法 ，也叫构造方法

构造器与方法的区别
构造器：使用new调用，用于实例化对象
方法：使用对象引用，是对象的业务功能

继承，封装，多态

继承：父类型 子类型

子类型共享了父类型的属性和方法，子类复用了父类的属性和方法

继承体现了多态：父类变量可以引用各种各样的子类实例
事务个体的多态：父类的子类实例是多种多样的
行为的多态：父类的方法被子类重写为多种多样
重载也是多种多样的
